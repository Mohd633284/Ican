📖 USAGE EXAMPLES:
------------------

Load Members from Cloud:
```javascript
import { getAllMembers } from '@/firebase';

const result = await getAllMembers('lagos');
availableMembers.value = result.data;
```

Save Invoice:
```javascript
import { saveInvoice } from '@/firebase';

await saveInvoice('lagos', {
  number: 123,
  customer: 'John Doe',
  amount: 50000
});
```

Upload File:
```javascript
import { uploadInvoicePDF } from '@/firebase';

await uploadInvoicePDF('lagos', 123, pdfFile, 
  (progress) => console.log(`${progress}%`)
);
```

Backup Data:
```javascript
import { backupLocalStorageToCloud } from '@/firebase';

await backupLocalStorageToCloud('lagos');
```
